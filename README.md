- `src/` — i sorgenti Markdown dei moduli: sono la fonte unica; la pagina HTML viene rigenerata da qui a ogni rilascio

Moduli Fondamenta (F1–F6) → Manuale operativo (M1–M22) → Corso AI Software Architect (A1–A15).
L'architettura completa è in `src/architettura-percorso-formativo-claude-v0.1.md`.
